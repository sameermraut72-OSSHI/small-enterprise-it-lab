# small-enterprise-it-lab
Enterprise IT infrastructure lab including networking, Active Directory, Microsoft 365, SharePoint, and Jira Service Management.
## Project Overview
This project demonstrates the design and implementation of a small office Information Technology setup inside a simulated environment for 30-50 user office. 
The objective was to build a realistic end-to-end business IT environment covering :
1) Enterprise network setup
2) Window Active Directory Domain Services
3) Microsoft 365 cloud identity management
4) Microsoft Entra ID administration
5) SharePoint employee self-service portal
6) Jira Service Management helpdesk ticketing
7) IT onboarding and support workflows

This lab simulates the type of infrastructure commonly used in small to medium-sized business environments.
